✨ Key Features
🧠
RL-Ready API
Standard reset/step interface compatible with any RL agent or LLM policy. Supports concurrent sessions with TTL management.
📊
Deterministic Grading
Transparent scoring from 0.0 to 1.0 with partial credit. Syntax checking, test-case execution, and reward shaping built in.
🎯
5 Difficulty Tiers
From simple syntax fixes to multi-bug nightmare scenarios. Progressive complexity tests true debugging reasoning.
🐳
Docker & HF Ready
Production-grade Dockerfile with non-root execution, health checks, and one-click Hugging Face Spaces deployment.
